# Employee-Salary-Analysis-using-SQL
Overview : This repository contains the answer scripts and queries for a structured 30-day SQL micro-course. The project focuses on solving practical SQL problems using a sample dataset (Hospital_Data) and demonstrates how to apply SQL for data analysis, aggregation, and reporting. It’s designed as a learning resource for beginners and intermediates who want to practice SQL queries with real-world scenarios.

Dataset : The course uses a table called Hospital_Data with the following fields:
          Hospital_Name 
          Location
          Department
          Doctors_Count
          Patients_Count
          Admission_Date
          Discharge_Date
          Medical_Expenses

This schema simulates hospital operations, making queries relatable to real-world analytics.

Key Queries & Concepts Covered : 
1) Aggregation : Total number of patients across all hospitals.
                 Average number of doctors per hospital.
2) Ranking & Filtering : Top 3 departments with the highest patients.
                         Department with the lowest number of patients.
3) Advanced Analysis :  Hospital with the highest medical expenses.
                        Average medical expenses per day per hospital.
                        Patients with the longest stay (date difference).
4) Grouping & Time Functions :   Patients treated per city.
                                 Average length of stay per department.
                                 Monthly medical expenses using EXTRACT(MONTH).
